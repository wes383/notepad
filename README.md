## Notepad

A minimal Markdown notepad with editing, preview, split view, and multi-format export.

## Features

- Markdown editing with preview
- Split view
- Syntax highlighting, GFM and math (KaTeX) support
- Theme toggle (light/dark)
- Editable title and character count
- Export to .md / .pdf / .png

## Local Development

```bash
npm install
npm run dev
```

## Build and Preview

```bash
npm run build
npm run preview
```
